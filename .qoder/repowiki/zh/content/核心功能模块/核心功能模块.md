# 核心功能模块

本系统按照业务领域划分为以下几个核心功能模块：

## 1. 用户系统与个人设置模块
负责应用的账号体系安全、权限控制和个人信息管理。
- **登录鉴权**：基于 SQLite 和 SHA-256 密码哈希加密的安全登录，配合 Vue Router 路由守卫实现强制登录拦截。
- **个人设置**：提供基本信息（用户名、头像上传预览）修改与严格的密码修改流程。
- **状态持久化**：使用 Pinia 记录当前登录用户的会话状态。

## 2. 云端文件上传模块
提供直连七牛云的本地极速上传能力，专为大文件与稳定性优化。
- **Node.js 流式上传**：绕过 Chromium 网络栈，直接使用原生 `http/https` 模块发起请求，有效解决大文件上传导致渲染进程内存溢出 (OOM) 的问题。
- **智能分片与断线重连**：对大于 4MB 的文件自动采用分片断点续传。内置网络异常自动退避与轮询恢复机制。
- **跨域与 CSP 解除**：解除了主窗口的同源策略限制，并放宽了 CSP，使前端能够直接加载并预览云端存储的图片和多媒体资源。

## 3. 本地记事本模块
提供基于富文本编辑器的本地笔记记录功能。
- 支持富文本格式编辑
- 基于 SQLite 的本地数据持久化
- 完整的增删改查交互

## 4. 接口测试工具模块
提供类似 Postman 的轻量级 API 测试功能。
- 支持多种 HTTP 请求方法 (GET, POST, PUT, DELETE)
- 支持自定义请求头和请求体
- 响应结果可视化展示与 JSON 格式化

## 5. 格式转换工具模块
提供多样化的本地格式转换能力。
- **图片转换**：支持 PNG、JPEG、WEBP、BMP、ICO 格式互转。
- **音视频处理**：基于本地 `fluent-ffmpeg` 的纯原生多媒体转换，内置浏览器原生播放器用于截取和裁剪精准的时间段。
- **JSON 格式化**：基础的 JSON 校验与美化。
- **高度组件化**：利用 Vue 3 Composition API (Hooks) 抽离了文件保存、时间大小格式化、媒体进度监听等通用逻辑。

## 6. 系统设置模块
管理应用的全局配置。
- 主题切换（亮色/暗色模式）
- 日志存储目录的配置与打开
- 转换工具的默认输出目录配置（支持后台静默转换，避免弹窗打扰）
