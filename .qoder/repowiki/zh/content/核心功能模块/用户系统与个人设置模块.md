# 用户系统与个人设置模块

## 概述

为确保应用数据和核心工具的访问安全，本应用内置了一套完整的本地用户系统。该系统实现了强鉴权路由拦截、密码单向加密存储以及基本个人信息（头像与用户名）的配置功能。

## 安全与鉴权设计

### 1. 密码加密存储 (Hash Encryption)

- 所有用户的密码**不以明文存储**。
- 在 `src/main/db.ts` 中，使用 Node.js 的原生 `crypto` 模块（SHA-256）对用户密码进行单向哈希加密后，再存入 SQLite `users` 数据库。即使用户拿到了数据库文件，也无法逆向解析出真实的密码。
- 系统初始化时，若检测到 `users` 表为空，将自动创建一个超级管理员账户（账号：`admin`，密码：`123456`）。

### 2. 路由守卫 (Route Guards)

- 配合前端的状态管理 `Pinia`（`src/renderer/src/store/user.ts`）持久化会话。
- 在 `src/renderer/src/router/index.ts` 配置了全局前置守卫（`router.beforeEach`）。
- **拦截策略**：对于任何未登录的访问请求，强制拦截并重定向到 `/login`。已登录用户尝试访问登录页则自动跳回主页。

## 个人设置 (Profile)

- **`Profile/index.vue`**：提供了一个两级 Tab 的设置中心。
- **基本信息**：
  - 支持直接将本地图片作为头像上传至云端（自动调用底层的七牛云流式上传 API `window.api.file.upload`）。
  - 上传成功后自动生成附带时间戳的网络 URL（避免浏览器强缓存），即时替换旧头像并同步至本地数据库。
  - 支持修改用户名。
- **安全设置**：
  - 支持验证原密码并修改新密码。修改密码成功后将强制注销（Logout）当前会话，确保安全性。
